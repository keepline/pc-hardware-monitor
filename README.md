# PC 硬件监控工具

一个现代化的PC硬件信息监控工具，采用黑色极简科技风格，实时显示您的计算机硬件信息。

## ✨ 功能特性

- 🖥️ **系统信息** - 显示操作系统、主板、BIOS等基础信息
- 🔥 **CPU监控** - 实时显示CPU使用率、温度、频率等
- 🧠 **内存监控** - 内存使用情况、DDR版本、频率、内存条详情
- 🎮 **GPU信息** - 显卡型号、显存、温度、使用率
- 💾 **存储信息** - 磁盘和分区使用情况
- 🌐 **网络信息** - 网络接口、IP地址、实时速度
- 🔋 **电池状态** - 电量、健康度、充电状态（笔记本电脑）
- 🎯 **快速导航** - 标签式导航栏，点击快速跳转到各个硬件模块

## 🎨 设计特点

- ⚡ 黑色科技风格，扁平化设计
- 🎯 彩色性能条，清晰区分不同组件
- 🏷️ 智能导航标签，快速定位到各个硬件模块
- 📊 实时数据更新（每5秒）
- 💫 流畅的动画效果
- 📱 响应式布局

## 🚀 快速开始

### 方法1：使用桌面快捷方式（推荐）

1. **首次安装依赖**（只需一次）：
```bash
powershell -ExecutionPolicy Bypass -Command "npm install"
```

2. **创建桌面快捷方式**：
```bash
powershell -ExecutionPolicy Bypass -File ".\create-shortcut.ps1"
```

3. **双击桌面的"硬件监控工具"图标启动应用**

### 方法2：命令行启动

#### 安装依赖

```bash
powershell -ExecutionPolicy Bypass -Command "npm install"
```

#### 运行应用

```bash
powershell -ExecutionPolicy Bypass -Command "npm start"
```

#### 开发模式（带DevTools）

```bash
powershell -ExecutionPolicy Bypass -Command "npm run dev"
```

#### 或使用批处理文件

双击 `start-monitor.bat` 文件即可启动

## 📝 使用说明

1. 启动应用后会自动加载所有硬件信息
2. 点击头部的**导航标签**（系统信息、处理器、内存等）快速跳转到对应模块
3. 滚动页面时，导航标签会自动高亮当前查看的模块
4. 点击右上角的"刷新数据"按钮可手动更新
5. 数据会每5秒自动刷新一次
6. 鼠标悬停在卡片上会有高亮效果

### 新增功能

- ✅ **桌面快捷方式** - 一键启动，无需命令行操作
